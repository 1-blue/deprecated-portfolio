# 👀 구현 기능
1. `scroll`에 의한 애니메이션
2. `accorion menu` 및 애니메이션 적용
3. `image-carousel`
4. 밤하늘의 별같은 움직이는 `background` ( using `<svg>` )
5. `TOC`

# 👇 가이드 라인
> 빌드 후 `/js/index.js`에 `./utils/index`부분을 `./utils/index.js`로 바꿔줘야 정상적으로 동작합니다.

+ 종속성 설치
```bash
npm install
```

+ 빌드
```bash
npm run build
```
